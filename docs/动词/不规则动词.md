# 不规则动词

### 规则变化

machen, macht, machte, gemacht

fragen, fragt, fragte, gefragt

重音在原形第一个音节，则第二分词有ge-，否则没有ge-。

- -ern结尾的动词是规则变化，比如erwidern, liefern ...
- -eln结尾的动词是规则变化，比如schütteln, klingeln ...
- -ieren结尾的动词是规则变化，比如passieren, reagieren ...
